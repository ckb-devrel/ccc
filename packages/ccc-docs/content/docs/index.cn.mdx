> CCC 是面向 CKB（Common Knowledge Base）区块链的一站式 TypeScript / JavaScript SDK，提供交易组装、多链钱包连接，以及 UDT、Spore、SSRI 等协议的集成能力。

## 你能用 CCC 做什么

- **学习 CKB** —— 丰富的代码示例与在线演示，帮你快速理解 CKB 的工作方式。
- **组装交易** —— 声明式、直观的交易组装，自动填充 Inputs 与手续费。
- **签名消息** —— 统一的签名接口，跨多个生态的钱包一致可用。
- **连接钱包** —— 开箱即用的钱包连接组件，或基于底层 Signer 构建自定义 UI。
- **分析链上数据** —— 以编程方式与 CKB 节点交互、处理链上数据。
