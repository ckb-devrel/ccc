---
title: 钱包集成
description: 为你的应用挑选合适的 CCC 钱包集成方式。
---

根据你的技术栈选择集成方式：

- **React** —— `@ckb-ccc/connector-react`，内置钱包连接弹窗。
- **Web Component** —— `@ckb-ccc/connector`，可用于 Vue、Svelte、原生 JS 等。
- **自定义 UI** —— 若需完全控制连接流程，使用 `@ckb-ccc/ccc`。
- **后端** —— Node.js 服务与脚本使用 `@ckb-ccc/shell`。
