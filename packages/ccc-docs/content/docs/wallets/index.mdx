Choose the integration that fits your stack:

- **React** — `@ckb-ccc/connector-react` with a built-in modal.
- **Web Component** — `@ckb-ccc/connector` for Vue, Svelte, vanilla JS, etc.
- **Custom UI** — `@ckb-ccc/ccc` if you want full control over the connect flow.
- **Backend** — `@ckb-ccc/shell` for Node.js servers and scripts.
