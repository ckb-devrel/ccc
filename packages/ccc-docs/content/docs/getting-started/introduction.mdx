CCC is a TypeScript/JavaScript SDK for building on the [CKB (Common Knowledge Base)](https://nervos.org/) blockchain. It gives you everything you need to interact with CKB nodes, compose transactions, sign messages, and connect wallets — across multiple chain ecosystems.

## What you can do with CCC

- **Learn CKB**: Explore code examples and live web demos to understand how CKB works hands-on.
- **Analyze data**: Query CKB nodes and process blockchain data programmatically.
- **Compose transactions**: Build transactions with an intuitive API — CCC handles input selection and fee calculation for you.
- **Sign messages**: Use a single signing interface that works across EVM, Bitcoin, Nostr, and more.
- **Connect wallets**: Drop in the connector component and go, or build a fully custom wallet UI.

## Key capabilities

- **Multi-chain wallet support**: One interface for wallets across EVM, Bitcoin, CKB, Nostr, and Dogecoin ecosystems.
- **Automatic transaction completion**: Describe your intent; CCC fills in the inputs and fee.
- **Unified signing**: The same `Signer` interface regardless of wallet type or chain.
- **React and Web Component connectors**: Pre-built UI components that work in React or any framework.

CCC powers production applications across the CKB ecosystem — including
