> CCC 是面向 CKB（Common Knowledge Base）区块链的一站式 TypeScript / JavaScript SDK，提供交易构建、多链钱包连接及协议集成（UDT、Spore、SSRI）能力。

## CCC 能做什么

- **了解 CKB**——通过代码示例和在线演示，理解 CKB 的工作原理。
- **构建交易**——声明式、直观的交易构建方式，自动补全输入和手续费。
- **对消息签名**——统一的签名接口，内置多链签名方法。
- **连接钱包**——开箱即用的钱包连接器，也可基于底层 `Signer` 自行构建自定义钱包连接界面。
- **分析链上数据**——以编程方式与 CKB 节点交互并处理区块链数据。
